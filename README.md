# AIprompt 🤖

A terminal-based Python app that uses the OpenAI API to generate random jokes and emoji-only replies.

## Features
- Generate random AI-powered jokes
- Get emoji-only replies from AI
- Live API calls using OpenAI's gpt-4o-mini model
- Simple and fast terminal interface

## How to Run

1. Clone the repo
```bash
git clone https://github.com/kazmi1012/AIprompt.git
cd AIprompt
```

2. Install dependencies
```bash
pip install openai
```

3. Add your OpenAI API key
```bash
export OPENAI_API_KEY="your-api-key-here"
```

4. Run the app
```bash
python main.py
```
